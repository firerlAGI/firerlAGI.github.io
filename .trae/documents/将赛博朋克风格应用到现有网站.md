## 迁移计划：将 Cyberpunk Portfolio 风格应用到 Astro 网站

### 第一阶段：样式系统迁移
1. 更新 Tailwind 配置，添加赛博朋克主题颜色（cyan-500、fuchsia-500 等）
2. 添加自定义字体
3. 添加自定义动画（glitch、pulse-fast、shimmer 等）
4. 创建全局样式文件，添加赛博朋克效果 CSS

### 第二阶段：核心组件转换
1. 将 React 组件转换为 Astro/React 组件
   - ParticleBackground（粒子背景）→ 保留为 React 客户端组件
   - GlitchText → 转换为 Astro 组件或保留 React
   - CyberCard → 转换为 Astro 组件
   - AiAssistant → 保留为 React 客户端组件（需要 @astrojs/react 集成）

2. 重构现有 Astro 组件
   - HeroSection → 采用赛博朋克 Hero 样式
   - ProjectsSection → 使用 CyberCard 风格
   - SkillsSection → 新增，使用进度条动画
   - ContactSection → 采用赛博朋克表单样式

### 第三阶段：页面整合
1. 更新主页布局，集成新组件
2. 添加粒子背景
3. 更新导航栏样式
4. 添加 AI 助手浮动按钮

### 第四阶段：功能集成
1. 安装必要的依赖：
   - @astrojs/react（用于 React 组件集成）
   - @google/genai（AI 助手）
   - lucide-react（图标）

2. 配置环境变量（GEMINI_API_KEY）

### 第五阶段：测试与优化
1. 响应式适配
2. 性能优化（粒子动画优化）
3. 主题切换兼容性检查
4. 构建测试

**预计影响范围**：
- 新增/修改：~15 个文件
- 核心改动：样式系统、主要板块组件
- 可选保留：GithubSection（根据需要调整）
